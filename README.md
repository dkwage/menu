# Kang's Bar Menu (강스 바 메뉴)

동적이고 고급스러운 스타일의 칵테일 메뉴 웹 애플리케이션입니다. 이 프로젝트는 베이스 주종별로 정리된 칵테일 목록을 반응형 디자인과 인터랙티브한 상세 보기 기능으로 제공합니다.

## 주요 기능

-   **동적 콘텐츠**: 메뉴 항목은 JSON 파일(`cocktails.json`)에서 동적으로 로드되므로, 코드를 수정하지 않고도 메뉴를 쉽게 업데이트할 수 있습니다.
-   **프리미엄 디자인**: 고급 바 메뉴에서 영감을 받아 필기체 폰트(`Great Vibes`, `Ephesis`), 부드러운 애니메이션, 깔끔한 레이아웃을 적용한 세련된 UI를 제공합니다.
-   **반응형 레이아웃**: 데스크톱과 모바일 환경 모두에 최적화되어 있습니다.
-   **인터랙티브 상세 정보**: 칵테일을 클릭하면 모달 창이 열리며 다음 정보를 확인할 수 있습니다:
    -   칵테일 이미지
    -   설명
    -   재료 목록
    -   도수 (ABV)
-   **카테고리별 표시**: 칵테일은 카테고리(시그니처, 보드카, 럼, 위스키, 진 등)별로 그룹화되어 표시됩니다.

## 기술 스택

-   **HTML5**: 시맨틱 구조.
-   **CSS3**: CSS 변수, Flexbox, Grid, 애니메이션을 활용한 커스텀 스타일링. 외부 CSS 프레임워크를 사용하지 않았습니다.
-   **JavaScript (Vanilla)**: 데이터 페칭(fetching), DOM 조작, 모달 로직 처리.
-   **JSON**: 칵테일 데이터(이름, 재료, 도수, 설명, 이미지) 저장.

## 프로젝트 구조

```
/
├── index.html       # 메인 진입점, 구조 및 모달
├── style.css        # 모든 스타일, 폰트, 반응형 디자인
├── app.js           # JSON 데이터 로드 및 메뉴 렌더링 로직
├── cocktails.json   # 메뉴 데이터 소스
└── images/          # 칵테일 이미지 디렉토리 (JSON에서 외부 링크 사용 시 불필요할 수 있음)
```

## 설치 및 사용법

1.  저장소를 클론하거나 파일을 다운로드합니다.
2.  최신 웹 브라우저에서 `index.html` 파일을 엽니다.
3.  메뉴를 업데이트하려면 `cocktails.json` 파일을 수정하면 됩니다.
